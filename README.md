git clone或下载zip本项目后，修改main.py 搜索#TODO按照注释修改即可。




```shell
#需要事先安装python环境
pip install -r requirements.txt

# 启动打卡（建议使用计划任务每天18：00启动 实现全自动）
python3 main.py
```
